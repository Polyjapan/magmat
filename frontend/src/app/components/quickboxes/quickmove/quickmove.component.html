Déplacer tous les objets de la catégorie

Utilisez le formulaire ci dessus pour rapidement déplacer des objets dans cet emplacement de stockage. Scannez les asset tag des objets à déplacer dans le champ de texte (un par ligne), et envoyez pour déplacer tous ces objets d'un coup.

Si vous souhaitez changer l'emplacement de tous les objets de la catégorie (et pas seulement l'objet scanné), cochez "Déplacer tous les objets de la catégorie".
